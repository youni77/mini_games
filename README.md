# Hangman Game (Tkinter)

A simple two-player Hangman game built with Python and Tkinter.

## Features

* Graphical User Interface (GUI) using Tkinter
* Two-player turn-based gameplay
* Random word selection from a predefined word list
* Separate chances for each player
* Word progress display
* Win and lose conditions
* Input validation for empty or repeated guesses

## How to Play

1. Run the program.
2. Players take turns guessing letters.
3. If a guessed letter exists in the word, it is revealed.
4. If the guess is wrong, the current player loses one chance.
5. The player who completes the word wins.
6. If both players run out of chances, the game ends and the word is revealed.

## Requirements

* Python 3.x
* Tkinter (usually included with Python)

## Run

```bash
python hangman.py
```

## Technologies Used

* Python
* Tkinter
* Object-Oriented Programming (OOP)

## نحوهٔ بازی

1. برنامه را اجرا کنید.
2. بازیکنان به نوبت یک حرف را حدس می‌زنند.
3. اگر حرف واردشده در کلمه وجود داشته باشد، در جای مناسب نمایش داده می‌شود.
4. اگر حدس اشتباه باشد، یک شانس از بازیکن کم می‌شود.
5. بازیکنی که کلمه را کامل کند برنده است.
6. اگر هر دو بازیکن شانس‌های خود را از دست بدهند، بازی تمام شده و کلمهٔ صحیح نمایش داده می‌شود.
